HTML中，总会用到js代码，但要追加一段代码，就要用到一些方式了。
我们可以用document.getElementById(“divid”).innerHTML来实现。使用方法：

//divid 是一个div标签的id 
var html = '你要插入的HTML代码';
document.getElementById("divid").innerHTML =  html;

示例：

<!doctype html>
<html>
<head>
<meta charset="utf-8"/>
<title>js发送内容到div</title>
</head>
<body>
<script>
var html = '发送成功';
document.getElementById("divid").innerHTML =  html;
</script>
<div id="divid"></div>
</body>
</html>
