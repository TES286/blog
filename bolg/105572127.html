前言

写网站不难，但很多人不想花那个钱，也就没有建自己的站。是可以把自己电脑做服务器，但不能确保24小时开机。一般一个网站下来几千块是有的。
但是如何只花8-9元建站呢，且听我娓娓道来。

平台

这里的平台是什么呢，那就是 Github 【狗头滑稽】
（惊不惊喜意不意外，没想到吧）（不过貌似速度有些慢，但一个私人网站应该够了【抠鼻.jpg】）

看到这，你是不是觉得我开玩笑？GitHub中全是代码，html写出来也没有用，顶多md文件可以看得到。
不，我们要用GitHub pages这样就可以任性的写网页了。

补充一下，GitHub pages没有服务器的概念，只能调用其他网站的API，自己的不行，因为其不支持asp或php等服务器脚本

自此预算为0

GitHub 多账号

这个就简单了，随便弄几个邮箱，就可以了，浏览器有记住密码的功能。

后面会讲到注册域名，可以有了域名之后到网页企业免费邮申请一下，加一个mx记录就可以了，域名填真的就行了，其他的随便填，能弄到200个账号（邮箱列表也可以算一个账号，也能用）

单纯文件可以可以用储存库，下载总有下载链接，html标签里直接拿过来用。

预算仍然为0

域名

这个大家应该都会弄，不要用top域名，要不然那200个邮箱是拿不到的，个人觉得site域名还好，比较便宜，在西数上注册8元，续费48元（年）

自此预算8元

配置

GitHub pages 官网的页面我制作了PDF，点击下载，这里面有配置的方法，cname到自己网站之后就可以直接访问自己的域名了（PDF看不清的点这里）

写在最后
